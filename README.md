**## API Endpoints

### Task Routes**
- **GET** `/api/tasks`  
  Retrieve the list of tasks.
- **POST** `/api/tasks/add`  
  Add a new task.
- **PUT** `/api/tasks/:id`  
  Update an existing task.
- **DELETE** `/api/tasks/:id`  
  Delete a task.

### User Routes
- **GET** `/api/users`  
  Retrieve the list of users.
- **POST** `/api/users/add`  
  Add a new user.
## Usage

### Adding a User
1. Navigate to the "Add User" section.
2. Enter the user's first and last name.
3. Save the user.

### Adding a Task
1. Navigate to the "Add Task" section.
2. Provide the task name, status, and assign a user.
3. Save the task.

### Editing a Task
1. Select a task from the task list.
2. Edit the task name, status, or user assignment.
3. Save the changes.

### Deleting a Task
1. Select a task from the task list.
2. Click "Delete" to remove the task.
